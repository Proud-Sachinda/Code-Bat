Batch Processing

A GUI interface to the generation of Gantt Chart schedules in Batch Process Optimisation from GAMS/CPLEX deterministic solvers.

Home How To About

Uploading a file
You can upload a file by dragging and dropping it on the space marked with a dashed border line. Only excel files may be uploaded in order to generate graphs.

Viewing the results
On the right side there is a section provided for you to view the results generated from your input file for your own satisfaction
You have an option to export the results to three different formats provided for you, there is PDF for portable document format, PPS for powerpoint presentation and PNG for a picture
