BATCH PROCESSING

Uploading a file

  • Drag a standard xlsx file for Batch Processing, into the input box.
  • Press the Upload button, after placing the xlsx file in the input box.
  • If the uploaded file is in the correct format, data will be displayed in the output section.
  • Choose the required format for the output, by pressing the Download As drop-down list.
  • Click the Export button to download the required output.
